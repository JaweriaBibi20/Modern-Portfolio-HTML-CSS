# 💻 Modern Personal Portfolio

A premium, high-class personal portfolio website designed to showcase my skills, projects, and academic background. 

## 🛠️ Tech Stack Used
* **HTML5:** For semantic and clean web structure.
* **CSS3:** For modern styling, custom gradient elements, responsive layouts (Flexbox/Grid), and smooth scroll effects.

## ✨ Features
* Fully responsive layout (mobile, tablet, and desktop friendly).
* Premium dark-themed design with smooth scrolling navigation.
* Clean and structured code showcasing basic frontend layout principles.
